## Project Overview
This **interactive web application** teaches users about **cyber safety** and safe online behavior.  
It covers key topics including **password safety, phishing awareness, social media privacy, and malware protection**.  
The app includes **practical tips, an interactive quiz, and a progress tracker** to make learning engaging and measurable.  

Created as part of my **portfolio for the KISP 2026 scholarship**, it demonstrates my **web development skills** and ability to build **dynamic, responsive educational applications**.  

---

## Features
- **Interactive Topics:** Explore cyber safety subjects with clear guidance.  
- **Tips Section:** Actionable advice for safe online practices.  
- **Quiz Section:** Test knowledge with multiple-choice questions and instant feedback.  
- **Progress Tracker:** Monitor completion and learning progress.  
- **Responsive Design:** Works on both desktop and mobile devices.  

---

## Technologies Used
- **HTML** – Page structure  
- **CSS** – Styling and layout  
- **JavaScript** – Interactivity and quiz functionality  

---

## How to Run
1. Download or clone the project folder.  
2. Ensure `index.html`, `style.css`, and `script.js` are in the same folder.  
3. Open `index.html` in a browser or code editor with live preview.  
4. Click topics to read content and complete the quiz.
